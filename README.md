# Step-motor-control-by-MFRC522-sensor-with-stm32-microcontroller-
In this video I have set the RC522 recorders
Then I defined a group of cards to run the step motor
MFRC522 PINS
-NSS->PB3
-SCK->PA4
-MOSI->PB5
-MISO->PB3
-VCC->3.3V
-RST->3.3V
-GND->GND
STEPPER MOTOR PINS
-VCC->5V
-GND->GND
-IN1->PE3
-IN2->PE2
-IN3->PE4
-IN4->PE5
